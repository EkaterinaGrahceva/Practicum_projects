### 📝 Краткое описание проекта

Проект представляет собой модульное тестирование Java-классов, моделирующих семейство кошачьих (Feline, Cat, Lion), а также дополнительного класса льва Алекса из мультфильма «Мадагаскар». Написаны тесты с использованием JUnit, Mockito и параметризации, достигнуто 100% покрытие кода (Jacoco), реализована инъекция зависимости для класса Lion.

### 🛠 Использованные навыки

- Написание модульных тестов на Java (JUnit 4)
- Создание мок-объектов с помощью Mockito
- Применение параметризации тестов (Parameterized tests)
- Оценка покрытия кода с помощью Jacoco (100% для целевых классов)
- Рефакторинг для внедрения зависимостей (Dependency Injection)
- Работа с Maven (подключение зависимостей, сборка)
- Написание тестов для наследуемых классов и переопределённых методов

### 📌 Вывод

Проект позволил на практике отработать техники модульного тестирования сложной логики с изоляцией зависимостей. Достижение 100% покрытия подтвердило тщательность проверки всех веток условий и методов. Дополнительное задание (класс Alex) продемонстрировало умение тестировать наследование и полиморфизм, а также писать параметризованные тесты для разных сценариев.
