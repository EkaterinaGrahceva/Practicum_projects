### 📝 Краткое описание проекта

Проект заключается в функциональном тестировании нового функционала бэкенда сервиса «Яндекс.Прилавка»: работа с наборами продуктов, расчёт доставки курьерской службой «Привезём быстро» и операции с корзиной (получение, добавление, удаление). Тестирование проводилось на основе документации API (Apidoc) с использованием Postman, по результатам составлен чек-лист и заведены баг-репорты.

### 🛠 Использованные навыки

- Анализ документации API (Apidoc)
- Тестирование API с помощью Postman (GET, POST, PUT, DELETE запросы)
- Формирование баг-репортов с описанием шагов, ожидаемого и фактического результатов
- Работа с тестовым стендом и переменными окружения
- Написание отчёта о тестировании

### 📌 Вывод

Проект позволил на практике применить знания по тестированию бэкенда: анализ требований, проектирование проверок и выполнение запросов к API. В ходе тестирования были выявлены дефекты в новой функциональности (наборы, доставка, корзина), что подтвердило необходимость тщательной проверки даже на уровне API до интеграции с фронтендом. Опыт работы с API Яндекс.Прилавка продемонстрировал умение самостоятельно тестировать серверную часть и готовность к задачам по автоматизации API-тестов в будущем.
