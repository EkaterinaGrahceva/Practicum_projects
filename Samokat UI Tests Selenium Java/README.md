### 📝 Краткое описание проекта

Проект представляет собой автоматизацию тестирования веб-сервиса «Яндекс Самокат» с использованием Java, Selenium и JUnit4. Написаны UI-автотесты для проверки работы выпадающего списка в разделе «Вопросы о важном» и позитивного сценария оформления заказа самоката с двумя наборами данных через разные точки входа (кнопки «Заказать» вверху и внизу страницы).

### 🛠 Использованные навыки

- Создание Maven-проекта и настройка зависимостей (Java 11, JUnit4, Selenium)
- Применение паттерна Page Object для описания страниц и локаторов
- Поиск и работа с веб-элементами (локаторы CSS, XPath)
- Написание автотестов с использованием JUnit4 (аннотации @Test, @Before, @After)
- Параметризация тестов (запуск одного теста с разными наборами данных)
- Кроссбраузерное тестирование (Google Chrome, Mozilla Firefox)
- Работа с ожиданиями (явные/неявные ожидания, WebDriverWait)
- Выявление и документирование бага (невозможность оформить заказ в Chrome)

### 📌 Вывод

В ходе проекта были успешно автоматизированы ключевые пользовательские сценарии сервиса «Яндекс Самокат», включая проверку отображения текстов в выпадающем списке и полный флоу заказа. Применение параметризации позволило сократить дублирование кода и повысить поддерживаемость тестов. В процессе выполнения был обнаружен критический баг в браузере Chrome, что подтвердило ценность автоматизации для раннего выявления дефектов.
