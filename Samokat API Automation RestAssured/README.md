### 📝 Краткое описание проекта

Проект представляет собой автоматизированное тестирование API учебного сервиса «Яндекс.Самокат» с помощью Java, RestAssured и JUnit4. Проверены основные ручки: создание курьера, авторизация курьера, создание заказа (с параметризацией цветов) и получение списка заказов, а также в дополнительной части — удаление курьера, принятие заказа и получение заказа по номеру.

### 🛠 Использованные навыки

- Автоматизация API-тестов на Java с использованием RestAssured
- Написание тестов с аннотациями JUnit4 (@Test, @Before, @After)
- Параметризация тестов (разные наборы данных для цветов самоката)
- Использование аннотации @Step для детализации шагов в отчётах Allure
- Генерация и публикация отчёта Allure (Allure Report)
- Обеспечение независимости тестов (создание и удаление тестовых данных)
- Проверка HTTP-кодов ответов и тела ответа (JSON)
- Работа с query-параметрами в RestAssured
- Написание негативных сценариев (ошибки валидации, несуществующие данные)

### 📌 Вывод

В ходе проекта были автоматизированы ключевые сценарии работы с курьером и заказами, включая позитивные и негативные проверки. Параметризация позволила эффективно протестировать создание заказа с разными комбинациями цветов, а отчёты Allure сделали результаты наглядными. Все тесты независимы (данные создаются и удаляются), что обеспечивает их стабильность и пригодность для регулярного прогона в CI/CD.
